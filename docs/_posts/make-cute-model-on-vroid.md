## 什么是Q版风格

> 以下是个人理解，并非专业定义

Q版（又称 cute 版）的模型主要是通过**简化人物细节**来达到**可爱**的效果，例如
+ 粗且缺少细节的手指
+ 几乎一般粗细的大腿、小腿、脚部
+ 头部细节很少但能反应出角色特点
+ 大眼睛
+ 简化着装

而 VRoid 的默认资产偏偏是细节较多的，加上素体模型更改了身体比例，所以大部分资产可能都需要自己去重新绘制纹理才能使用。

核心是简化，通过卡通般的人物比例，让人们普遍觉得这个角色可爱。

## 具体步骤

拿到了素体模型之后，最要紧的是先把角色的发型给做出来。这步最为繁琐，因为 VRoid 的默认发型都是细节极高的，做出来的效果不是很 cute。

> 不要急着操作，我建议你看完这篇文章再去操作。

点击「发型 -> 刘海」选项，然后在右侧点击「编辑发型」开始手动搓发型，你也可以先选好心仪的发型，调好颜色再进行修改。  
修改发型是很繁琐的一件事，如界面所示，有什么东西可用已经告诉你了，你只需要上手试一试就知道要怎么编辑发型了，界面操作还是很直观的，就是改起来繁琐。

![](https://pic1.imgdb.cn/item/69dba318d93192251ec48827.webp)

修改发型有一个很重要的点是修改纹理，默认纹理是有预制的高光的，这个高光很不 cute，需要把它去掉。  
具体位置在编辑发型界面中先选中一个头发，然后再在右侧展开显示为亮色的材质，把“高光颜色”给关掉  
![](https://pic1.imgdb.cn/item/69db9de9d93192251ec48696.webp)

改完发型之后，然后是修改嘴巴颜色，先到「面部 -> 编辑表情」调一个会张嘴的表情，然后修改「嘴巴内部」，随便选一个自己喜欢的，这个倒是无所谓。

再然后就是眼睛了，大头大眼睛才更贴合Q版模型。VRoid 自带的眼睛预设也是细节很高的，需要自己去重新调节眼睛大小，并且修改纹理。  
我的眼睛配置如下所示，仅供参考：（参数是可以超出滑条范围的，手动输入就行了）
- 眼睛横长: `1.500`
- 眼睛纵长: `2.700`
- 眼睛位置: `-0.029`
- 眼距: `-0.817`

一般需要修改两个地方的纹理，「眼睛」和「眼睛高光」，你可以参考一些Q版插画的眼睛来修改纹理，总之越简洁的眼睛，就越有感觉。  
我就按我 OC 的设定，使用上下硬渐变颜色，并添加类似 fumo 玩偶的左上角白色圆圈高光。

![](https://pic1.imgdb.cn/item/69db9e52d93192251ec486b7.webp)

如你所见，我复用了之前的服装，因为上次制作的这套服装稍微改改比例还能穿得上，就先用着了。

我还顺带编辑了一些表情，让表情内敛一点，默认的预设表情太过于夸张了，修改过程就不多赘述了。

实际上，我制作的发型偷懒了，只有「刘海」是自己做的，「后发」和「呆毛」用的是预设模板，所以看起来细节比较丰富。理想情况下，应该减少一点头发，让每一片头发的宽度大一点，使得发型更简洁。

> Overall, sir. Your model will cute.

## 修改人物比例是怎么实现的

通过实际使用这个素体模型发现，在「服装」板块有个「上半身内衣」，这个素体模型就是用「上半身内衣」来实现的。所以，**不要在服装中直接应用全身套组**，否则会覆盖掉「上半身内衣」，导致人物比例**失效**。

而且，如果你要修改人物的皮肤颜色，需要修改以下内容
+ 面部 -> 皮肤 -> 编辑纹理
+ 体型 -> 全身 -> 编辑纹理
+ 服装 -> 上半身内衣 -> 编辑纹理 -> 连体衣（用「油漆桶」工具修改那两个图层的颜色）

另外，因为通过这个方法来修改人物比例，一些服装将会无法直接使用，例如
+ 腕部装饰（主要是手套）
+ 腿部服饰（用于修改比例的「上半身内衣」会挡住丝袜）
+ 鞋（脚变大了，所有鞋子都不能用，而且插画风格的高细节鞋子也不太搭Q版模型）

鞋子这块，目前我没有太好的解决方法，我只填了个纯黑色的色块上去先顶着（随便找个能缩放的鞋子，然后改纹理为纯黑色）。所以，要么就光脚，要么就简单整整。

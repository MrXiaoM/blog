在高中之后，不管你上本科还是专科都用得上的一些东西。在开学之前就囤好，可以省不少麻烦。算是踩坑记录总结了。

开学之前，班级助理（由学长担任）会挨个联系学生，拉你到班级的微信群，通常会在群里提供快递地址。最好将学校的快递地址存到微信收藏里面以免遗失，临近开学就通过电商平台购买物品、寄到学校。

到学校办好手续，有自由时间之后，第一件事就是逛遍校园的每一个角落，至少要知道快递驿站在哪里。但是快递驿站也可能会变动，如果哪个学期开始找不到快递驿站了，通过抖音、小红书等主流平台（虽然我不喜欢）也许能够获取学校新的快递驿站位置。

这份指南仅适用于南方、上床下桌的四人间，宿舍有独立卫生间，卫生间有热水提供的情况。旨在帮助你在校园里生活得更好、更顺利，提升生活质量。

## 基础物品

这些是需要你和舍友集资购买的物品
+ [ ] 洗衣机：刚开学大概率会有学长来推销，买就是了
+ [ ] 洗衣液：给洗衣机用的洗衣液，可以跟舍友商量集资购买，或者由某人承担购买洗衣液的工作
+ [ ] 防虫网：安装在阳台上，避免什么杂七杂八虫子飞进来。如果你是在一楼，完全可以找宿管借梯子自己安装，在高层比较麻烦一点
+ [ ] 拖把桶和拖把/扫把/垃圾铲：清洁用具，建议去和舍友商量每周末轮值清理一次宿舍，保持干净整洁
+ [ ] 洁厕灵：定期购买，清理厕所使用
+ [ ] 饮水机/桶装水：学校可能会发饮水机，并提供桶装水送货上门服务，

宿舍必须拉一个微信群，使用群收款工具可以很方便地筹集购买消耗品的钱。然后分配指定由某人来定期购买某样东西，例如桶装水、电费等等。

## 生活用品

忽略被褥、牙膏牙刷、毛巾、衣架、洗衣粉、洗发水等这些生活必需品。

+ [ ] 桶：大概率会有学长来推销洗衣机，除了开学那几天，不太用得上桶，不过带上还是保险的
+ [ ] 盆：洗贴身衣物用，内裤袜子不要放进洗衣机
+ [ ] 贴片镜子：可以贴在自己的柜门上、洗手台上，方便整理仪容。不过贴片有一个缺点，容易弯曲，贴在不平整表面上的话，稍微不注意一用力，容易变成哈哈镜
+ [ ] 床帘：通常是床帘跟蚊帐一体的，给自己留点隐私空间
+ [ ] 泡面碗：如果你要吃泡面，就买一个。不过通常学校食堂都支持外卖，要么就是用自建小程序外卖平台，要么就是用主流外卖平台，都是学生校内配送。点惯了外卖之后，外卖比泡面方便，泡面也就偶尔吃。
+ [ ] 保鲜袋：可选，有需要的时候可能会用到
+ [ ] 纸巾：一些平台活动可以一块钱白嫖，不过就算是正常从电商平台买，十几块钱也够自己一个人用半年了
+ [ ] 垃圾袋：在宿舍产生的垃圾总要有东西装，你不一定有时间每天出去获得购物袋来充当垃圾袋，买几卷垃圾袋是必要的。垃圾袋这块有大坑，第一次买不要贪便宜，便宜的可能会渗水渗油、不防臭。各个品牌都试试，最后只买自己觉得好用的品牌

以下是一些电器类的物品
+ [ ] 烧水壶：选一个低功耗的烧水壶，通常比较低的都有 400W，宿舍限电的话只要不是多个大功率电器同时使用就没事。烧热水喝或者做泡面都方便一点，虽然宿舍可能会配饮水机，但是温度可能会不够
+ [ ] 排插：或者插排、插线板，不管怎么说，都是一个东西。宿舍通常只会一个床位给一个二插+三插，甚至更少。不需要买太多孔位的，限制孔位数量就是为了限制功耗，避免超出电网承载能力出现事故。
+ [ ] 挂钩：购买一些粘钩，将物品挂在柜子上，可以有效节省空间。另外购买一个挂在高处的铁钩子，挂在床架上，用来放置自己的背包
+ [ ] USB分线器：给一些小 USB 设备提供 5V1A 供电而只占用一个二插电源接口，下面我会给一些用例
+ [ ] 灯管：电商营销叫法好像叫什么“酷毙灯”，总之是 USB 供电的长条灯管。宿舍的光照可能是不够的，大概率只有后面通向阳台的那面有较大的窗户，白天都可能需要开灯，而且开灯也不一定有足够的光照能给你写字什么的，所以需要额外加装一个灯管提供光照。
+ [ ] 时钟：带电池的 USB 充电时钟。虽然你可以用手机和电脑看时间，有一个可以随时看到时间的设备也还是挺重要的。
+ [ ] 温度计/湿度计：纯摆件，看你想不想要

以下是其它工具类物品
+ [ ] 挖耳勺：不多解释
+ [ ] 指甲钳：不多解释
+ [ ] 虎头钳：需要暴力改装某些东西，比如拆除支架上的一条不必要的横杠时需要使用
+ [ ] 螺丝刀套装：需要拧点什么的时候，比如给笔记本加装硬盘，会用得上的

## 文具

以下这些虽然有一些可能不太用得上，但是我认为都是缺一不可的。特别是学校在犄角旮旯的工业区这种情况，你急用都没地方能很快地买到文具。

+ [ ] 签字笔：黑色、红色，其它颜色可以有但没必要
+ [ ] 常规铅笔/机考2B铅笔：在正规考试会有用到的时候的
+ [ ] 液态胶水/固态胶水：有一些资料、二课活动有可能需要用胶水
+ [ ] 剪刀：同上，平时需要裁剪点什么东西也用得上
+ [ ] 订书机：平时用不上，但是需要装订资料的时候找不到，所以最好备一个
+ [ ] 尺子套装：一些学科可能会要求你画图，就能用上尺子了
+ [ ] 双面胶/透明胶带/电工胶布：可选，用的情况不多，更多时候是自己要用而不是学业需要

## 硬件相关

大多围绕电脑、网络进行。一些必要的硬件。
+ [ ] 笔记本电脑：专业需要也好，玩游戏也好，有一台笔记本电脑总是好的。毕竟高配置台式机是电老虎，也不方便带走，笔记本电脑比较合适
+ [ ] 移动硬盘：比闪存盘（俗称U盘）要耐用很多，速度也快很多，可以用硬盘盒方案来做
+ [ ] 笔记本电脑支架：对于放在宿舍不移动的情况，我更推荐使用烧烤架支架，皮实耐用，节省空间。我电脑包里还放了另一个便于移动的折叠支架以防万一
+ [ ] 拓展坞：笔记本接口通常不多，但是一个键盘+一个鼠标就占了两个 USB 接口，指定是不够用的，要用扩展坞增加接口
+ [ ] 耳机：尽可能不要在宿舍里外放，以及不要自带麦克风的耳机，那玩意是累赘
+ [ ] 摄像头/麦克风：可选，需要视频通话、直播的时候才购买，不要指望笔记本自带摄像头的质量
+ [ ] 闲置手机：是的，避免自己的手机出现问题，你最好要备至少一台旧手机，以免出现紧急情况无法处理
+ [ ] 软路由：可选，看你的校园网环境，我的校园网我嫌登录麻烦，还限制设备。不过限制不严，就弄了个软路由，通过重设 MAC 实现多设备共享校园网

## 违禁物品

像是电磁炉、电煮锅之类的大功率电器不要买，包括厨具也不要买，宿舍里不能做饭，宿管/辅导员来了一收一个准。而且宿舍限电，你一个人开了这些电器，其他人就用不了电了，自动跳闸。

宿舍通常不允许养宠物，至少我学校不允许。

其它的暂时没想到，如果你不确定的话，可以去网上搜一下能不能放进宿舍，大多数学校的标准是相同的。

## 其它坑

+ 注意空调的能耗，我宿舍的空调非常耗电，非必要情况都不舍得开，电费嗷嗷掉
+ 先去了解食堂都有哪些商家入驻，以及安装好外卖软件，你都上大学了，不要保持高中的习惯，吃点好的
+ 刚来学校肯定不太熟悉环境，把校园平面图常驻保存到相册，方便查看
+ 即使学校推荐，不要使用任何有广告的课表软件。说的就是你超级课程表，看课表前还要看广告，纯恶心自己
+ 如果有校园跑可以用自行车来物理偷鸡，什么摆手摇晃检测只要找到技巧很容易绕过
+ 如果体育课是能够自选的，定向运动/定向越野恐怕是最简单的体育课程，在校内只需要跑遍整个校园的固定点位就可以了
